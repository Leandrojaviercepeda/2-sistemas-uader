Nombre del programa: semaforo-app , Version: 1.0, Fecha actual: 17/01/2019
-----------------------------------------------------------------------------

Descripcion breve del programa: esta aplicacion simula una interseccion de 4 calles en cada una de las cuales hay un semaforo.

-----------------------------------------------------------------------------

Requisitos basicos para la instalacion:  una vez descomprimido el archivo, localice el archivo ejecutable "semaforo-app" ubicado en la carpeta "/dist" y comience a utilizar la aplicacion.
Recursos esperados: SO linux 64 bits, 600 mb de disco, 500 mb RAM.
